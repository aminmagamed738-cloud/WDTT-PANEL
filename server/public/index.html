Запустите туннель чтобы получить конфиг

  1. Убедитесь что outbound с тегом wdtt-socks5 уже добавлен (вкладка SOCKS5)

  2. Нажмите Save → Restart Xray

  1. После настройки проверьте что трафик идёт через прокси (выполните на VPS):

Проверка SOCKS5 
curl --socks5 127.0.0.1:PORT https://2ip.ru
Проверка HTTP 
curl --proxytunnel -x http://127.0.0.1:PORT https://2ip.ru

  1. Если команда вернула IP-адрес — прокси работает ✓

  2. В самой 3x-ui: нажмите ⋯ рядом с нужным Outbound → Test
